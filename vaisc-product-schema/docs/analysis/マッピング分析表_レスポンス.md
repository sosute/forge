### まとめ割メッセージ生成（数値型活用）

```javascript
// JSON数値型による精密な処理
const bulkDiscountMessage = (() => {
    const minQty = product.attributes
      .find(attr => attr.key === "bulk_discount_min_qty")
      ?.value?.numbers?.[0];
    
    const rate = product.attributes
      .find(attr => attr.key === "bulk_discount_rate")
      ?.value?.numbers?.[0];
    
    if (minQty && rate) {
        return `${minQty}個以上で${rate}%OFF`;
    }
    return null;
})();
```
